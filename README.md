# BMI-Calculator
📝 Program Workflow:

⚖️ First, the program asks the user to enter their weight.

📏 Then, it asks the user to enter their height.

🧮 The program calculates the BMI using the BMI formula and displays the result.

---------------------------------------------------------------------------------------------

🔁 Replay Option:

✅ If the user selects Yes, the program asks for new input values again.

❌ If the user selects No, the program terminates
